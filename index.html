🎬 Jeden krok do pełnej wersji

Moduł „Co obejrzeć" korzysta z bazy TMDB. Potrzebujesz darmowego klucza API — wpisujesz go raz, zapisuje się w Twojej przeglądarce.

  1. Wejdź na themoviedb.org/settings/api
  2. Kliknij Create → Developer
  3. Wypełnij formularz (dowolna nazwa, np. "Daily Helper")
  4. Skopiuj API Key (v3 auth) i wklej poniżej
🔒 Klucz zapisuje się wyłącznie w Twojej przeglądarce (localStorage). Nikt inny go nie widzi. Możesz go usunąć w każdej chwili klikając ⚙️ w nawigacji.
