Darmowa kolekcja
Bez rejestracji. Starannie dobrana baza ~80 filmów i seriali z ocenami i platformami streamingowymi.
🌐
Baza TMDB
Wymaga darmowego klucza API. Dostęp do miliona tytułów, aktualnych plakatów i platform streamingowych.
🔑 Wpisz klucz API TMDB
  1. Wejdź na themoviedb.org/settings/api
  2. Kliknij Create → Developer
  3. Wypełnij formularz (dowolna nazwa, np. "Daily Helper")
  4. Skopiuj API Key (v3 auth) i wklej poniżej
🔒 Klucz zapisuje się wyłącznie w Twojej przeglądarce. Nikt inny go nie widzi.
